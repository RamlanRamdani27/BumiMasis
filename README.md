#  Getting Started

### Prerequisites

-   [Git](https://git-scm.com/)
-   [PHP](https://php.net/)  php => ^8.0.x
-   [Laravel](https://laravel.com/docs/9.x)  laravel => 9.x
-   [Composer 2](http://getcompose.org/)  (`composer install`)
